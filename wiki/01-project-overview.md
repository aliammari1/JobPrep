# Project Overview

## 🎯 What is JobPrep AI?

JobPrep AI is a comprehensive, AI-powered career preparation platform designed to help job seekers ace interviews, build professional CVs, master coding challenges, and stand out in today's competitive job market.

## 🌟 Vision & Mission

### Vision
To democratize access to high-quality interview preparation and career development tools, making them available to everyone regardless of their location or financial situation.

### Mission
Empower job seekers with AI-powered tools and insights that:
- Build confidence through realistic practice
- Improve technical and soft skills
- Create professional, ATS-optimized application materials
- Provide actionable feedback for continuous improvement

## 🎯 Core Value Propositions

### For Job Seekers
1. **Realistic Interview Practice** - Practice with AI interviewers that simulate real hiring managers
2. **Instant Feedback** - Get detailed analysis of your answers, tone, and body language
3. **Professional CVs** - Create ATS-optimized resumes that get past automated screening
4. **Coding Practice** - Master technical interviews with multi-language code challenges
5. **Comprehensive Preparation** - All-in-one platform for every aspect of job search

### For Educational Institutions
1. **Career Center Enhancement** - Modern tools to support student career development
2. **Scalable Solutions** - Support unlimited students without additional staff
3. **Analytics** - Track student progress and identify areas for support
4. **Industry Preparation** - Bridge the gap between education and employment

### For Corporate Training
1. **Interview Training** - Prepare employees for internal promotions or role changes
2. **Skills Assessment** - Evaluate technical skills objectively
3. **Onboarding** - Help new hires prepare for technical assessments

## 🏗️ Core Components

### 1. Mock Interviews
AI-powered interview simulation with real-time video, voice analysis, and intelligent feedback.

**Key Features:**
- Multi-AI support (Gemini, GPT-4, Claude, Ollama)
- Real-time video conferencing
- Behavioral and technical questions
- Body language analysis
- Detailed feedback reports

### 2. CV Builder
Professional resume builder with ATS optimization and LinkedIn integration.

**Key Features:**
- Three professional templates
- LinkedIn auto-import via Chrome extension
- AI-powered suggestions
- Multi-format export (PDF, DOCX)
- Version control

### 3. Code Challenges
Multi-language coding practice platform for technical interview preparation.

**Key Features:**
- 15+ programming languages
- Difficulty-based problems
- Real-time code execution
- Test case validation
- Leaderboard and statistics

### 4. Cover Letter Generator
AI-powered cover letter creation tailored to job descriptions.

**Key Features:**
- Job description analysis
- Personalized content
- Multiple AI providers
- Customizable tone
- Quick revisions

### 5. Chrome Extension
One-click LinkedIn profile import for instant CV creation.

**Key Features:**
- Automatic data extraction
- Privacy-focused (local processing)
- Seamless integration
- Regular updates

## 🛠️ Technology Stack

### Frontend
- **Next.js 15.5** - React framework with App Router
- **React 19.1** - Modern React with Server Components
- **TypeScript 5** - Type safety throughout
- **Tailwind CSS 4.0** - Utility-first styling
- **shadcn/ui** - Accessible component library

### Backend
- **Node.js 20+** - JavaScript runtime
- **Prisma 6.17** - Type-safe database ORM
- **PostgreSQL** - Reliable relational database
- **Better Auth 1.3** - Modern authentication

### AI & ML
- **Google Gemini** - Primary AI provider
- **OpenAI GPT-4** - Advanced reasoning
- **Anthropic Claude** - Alternative AI
- **Ollama** - Local AI models
- **HeyGen** - AI avatars
- **MediaPipe** - Body language analysis

### Real-time
- **LiveKit** - WebRTC video/audio
- **Socket.io** - Real-time notifications
- **Liveblocks** - Collaborative features

### Infrastructure
- **Vercel** - Deployment and hosting
- **Appwrite** - File storage
- **Cloudinary** - Image optimization
- **Stripe** - Payments
