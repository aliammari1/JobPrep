## How to Create Real Screenshots

1. **Using Browser DevTools**:

   - Open your app in Chrome
   - Press F12 to open DevTools
   - Click device toolbar (Ctrl+Shift+M)
   - Set viewport to desired size
   - Take screenshot using DevTools

2. **Manual Capture**:

   - Set your browser to exact dimensions
   - Navigate to main screens
   - Use screenshot tools (Cmd+Shift+4 on Mac, Win+Shift+S on Windows)

3. **Automated Tools**:
   - Use Playwright/Puppeteer for consistent screenshots
   - Capture at exact resolutions

## Best Practices

✅ Show actual app interface
✅ Use high-quality, clear images
✅ Include both light and dark mode if supported
✅ Show key features and functionality
✅ Use consistent branding
